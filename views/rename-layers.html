Rename layers

Format

Note:
  • Use {seq} to insert the sequence number starting from 1.
  • Use {idx} to insert the sequence number starting from 0.
  • Use {name} to insert the original layer name.
  • Use {count} to insert the total number of layers.
  • Use {type} to insert the type of the layer in lowercase, i.e: frame, text, etc.
  • Use {type-uppercase} to insert the type of the layer in uppercase, i.e: FRAME, TEXT, etc.
  • Use {type-capitalized} to insert the type of the layer in capitalized, i.e: Frame, Text, etc.
